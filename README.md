# s2gx-2015-iot-demo-hub
Demo application for Booting IoT with Grails at SpringOne2GX 2015

#Demo structure
* Grails 3
* Raspberry Pi 2
* Phillips Hue
* Fadecandy
* Neopixel matrix
* Processing Application
* SMS (Twilio)
* XBee over Serial Device
